TRACKABLE EVENTS

  • → page_view — on load
  • → click — any button / link
  • → form_submit — form submission
  • → scroll_depth — 25/50/75/100%
  • → time_on_page — every 30 s
  • → product_click — card interaction

HOW IT WORKS

The analytics.js script batches events every 2 s and sends them to the Node.js backend via POST /track. The backend stores them in SQLite. The dashboard polls GET /analytics every 5 s to stay fresh.
